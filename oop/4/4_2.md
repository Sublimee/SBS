**Полиморфизм подтипов**

Есть один интерфейс (foo()) и разные его реализации (в классах Cat и Bird). Вызываемый метод выбирается исходя из фактического типа объекта, а не типа в объявлении.

**Параметрический полиморфизм**

Одна и та же функция с одним и тем же кодом способна работать с разными типами:

```python
def length(x):
    return len(x)
```
