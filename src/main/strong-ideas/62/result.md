Код представляет собой вычисления ряда Фибоначчи с помощью корутины fibs_sum. При каждом вызове gsum.send(b) к сумме fsum добавляется b, это обновленное значение суммы возвращается. 

# 2

```python
def fibs_sum():
    fsum = 0
    while True:
        fsum = yield fsum # ошибка заключась в занесении в последовательность суммы всех предыдущих элементов вычисленных элементов ряда вместо очередного значения ряда, т.е. суммы текущего и предыдущего значений ряда
     
def get_fibs(num):
    a, b = 0, 1
    gsum = fibs_sum()

    gsum.send(None)
    for i in range(num):
        yield gsum.send(b)
        c = b
        b = a + b
        a = c
```
