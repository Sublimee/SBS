# 1

### 1.1

Поменять a и b на любые другие, которые удовлетворяют предусловию:

```bash
{ true }
a = 2;
{ a > 1 }
b = 2;
{ a > 1, b > 1 }
c = a + b;
{ c > 1 }
```

### 1.2

Поменять сложение на умножение:

```bash
{ true }
a = 2;
{ a > 1 }
b = 2;
{ a > 1, b > 1 }
c = a * b;
{ c > 1 }
```

### 1.3

Сделать зависимость b от a:

```bash
{ true }
a = 2;
{ a > 1 }
b = a;
{ a > 1, b > 1 }
c = a + b;
{ c > 1 }
```

### 1.4

Поменять выражение для вычисления c:

```bash
{ true }
a = 2;
{ a > 1 }
b = 2;
{ a > 1, b > 1 }
c = 2 * a + 2 * b;
{ c > 1 }
```
