В моем решении стоило упомянуть, что классы организовываются, в том числе с помощью иерархии каталогов. Это очевидно, но явно прописано не было. Здесь же стоит вспомнить, что классы могут находиться в пакете без имени (default package).

Вероятно, в дополнение к иерархии каталогов стоит упомянуть еще и модули, которые появились в Java 9, которые в свою очередь тоже организованы с помощью иерархии каталогов.

Также мне нелишним было бы указать синтаксис для подключения пакета и для импорта.
