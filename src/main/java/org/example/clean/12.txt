№1 Связывание во время компиляции
Настройки для кнопки кручения барабана берутся из конфиг-файла. Это сделано для того, чтобы при изменении текста на
кнопке не пришлось делать перепоставку кода микросервиса, а просто подменить конфиг.

№2 Связывание во время написания
Идентификаторы программ повышенного кэшбэка заданы константами. Они не будут изменяться (только добавляться новые). Их
бессмысленно задавать через конфигурацию, так как добавление новой программы влечет за собой написание логики по ее
обработке.

№3 Связывание во время выполнения
Некоторые ответы микросервис формирует в зависимости от входящих параметров из запроса пользователя. Например, отдает
список карт пользователя.
